World-Class Website Development Prompt for Dr. Martin Lemelle
Project Overview
Create a presidential-level, world-class website for Dr. Martin Lemelle, Jr., 11th President of Grambling State University. This website should exceed the quality and sophistication of the White House website, showcasing modern design principles, seamless user experience, and technical excellence.
Design Philosophy & Principles
Visual Hierarchy & Brand Identity

Color Palette: Gold (#FFD700, #F4C430), White (#FFFFFF), Black (#000000)
Typography: Use elegant serif fonts for headings (Georgia, Times New Roman) and clean sans-serif for body text (Helvetica, Arial)
Contrast: Ensure WCAG 2.1 AA compliance with proper contrast ratios
Sophistication: Channel executive presence through refined, minimalist design

Modern UI/UX Principles

Progressive Enhancement: Mobile-first responsive design
Micro-interactions: Subtle animations that enhance user experience
Performance: Sub-2 second load times with optimized assets
Accessibility: ARIA labels, keyboard navigation, screen reader compatibility
Visual Storytelling: Use white space, typography, and imagery to guide user journey

JavaScript Functionality
javascript// Intersection Observer for Animations
const observerOptions = {
  threshold: 0.1,
  rootMargin: '0px 0px -50px 0px'
};

const observer = new IntersectionObserver((entries) => {
  entries.forEach(entry => {
    if (entry.isIntersecting) {
      entry.target.classList.add('animate-in');
    }
  });
}, observerOptions);

// Observe all animation elements
document.querySelectorAll('.animate-on-scroll').forEach(el => {
  observer.observe(el);
});
